[−][src]Module dynomite::retry

Retry functionality

Specifcally this implementation focuses on honoring these documented DynamoDB retryable errors on top AWS's general recommendations of for retrying API requests.

examples

 use dynomite::{Retries, retry::Policy};
 use dynomite::dynamodb::{DynamoDb, DynamoDbClient};

 let client =
    DynamoDbClient::new(Default::default())
        .with_retries(Policy::default());

 // any client operation will now be retried when
 // appropriate
 let tables = client.list_tables(Default::default());
